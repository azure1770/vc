# VC

VC is a Script to a Mount VeracryptContainer and open a KeepassXc Database from inside of the Container

## Installation


```bash
git clone https://github.com/azure1770/vc
```
## Post Install
edit /bin/vcvars to your needs.

## Usage
vc [option]

vc help
