# VC

VC is a Script to a Mount VeracryptContainer and open a KeepassXc Database from inside of the Container - For Linux only!

## Installation


```bash
git clone https://github.com/azure1770/vc.git
cd vc
chmod +x install.sh
./install.sh
```
## Post Install
edit /bin/vcvars to your needs.

## Usage
vc [option]

vc help
